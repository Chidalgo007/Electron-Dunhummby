# ⚡ Automated Power BI Data Pipeline  

This project is a fully automated **three-step data processing workflow** designed to streamline repetitive manual tasks and ensure timely updates to **Power BI dashboards**.  

---

## 🔹 Step 1 – Automated File Retrieval  
🔑 Logs into a secure data server  
🗂️ Navigates the interface  
⬇️ Automatically downloads raw data files  

👉 This eliminates manual clicks, reduces human error, and saves significant time.  

---

## 🔹 Step 2 – File Upload and Remote Processing  
⚙️ Locally processes and transforms the raw data  
📤 Uploads files back to the server for further processing  
⏳ Wait time: ~3 hours (completely automated)  

👉 Ensures correct formatting and submission without user intervention.  

---

## 🔹 Step 3 – Processed File Retrieval and Integration  
⬇️ Automatically downloads processed files  
📦 Unzips and organizes into the correct folder structure  
📊 Makes datasets immediately available in **Power BI**  

👉 Provides stakeholders with **real-time, up-to-date insights**.  

---

## 🛠️ Technology Stack  
- 🚀 **Electron + JavaScript** → Cross-platform interface & web automation  
- 🐍 **Python** → High-performance Excel manipulation  
- 🔗 **Hybrid Integration** → Combines **browser automation** with **backend data processing** in one executable  

---

## 🎯 Impact  
By merging **web navigation, file management, and data transformation** into one cohesive tool, this program transforms a **multi-hour, error-prone process** into a **hands-free, efficient pipeline**—freeing up time for higher-value tasks.
